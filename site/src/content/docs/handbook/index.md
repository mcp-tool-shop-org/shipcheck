## What is Shipcheck?

"Done" used to mean the code works. That's not enough. A product is code + safety + error handling + docs + identity + shipping hygiene. Shipcheck defines the bar with 27 hard gates that block release and 4 soft gates that define "whole."

The gate says **what** must be true, not **how** to implement it. Applicability tags prevent checkbox shame on repos where items don't apply.
