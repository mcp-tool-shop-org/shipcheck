# Security Policy

## Scope

Shipcheck is a standards repository containing markdown templates, contracts, and adoption guides. It contains **no executable code**, collects **no data**, and makes **no network requests**.

## Reporting

If you find a security issue in the standards themselves (e.g., guidance that would lead to insecure implementations), please report it to:
